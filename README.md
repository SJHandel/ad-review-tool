## üöÄ How to Use

### First-Time Setup (5 minutes)

1. **Install Node.js**
   - Go to https://nodejs.org/
   - Download and install the LTS version
   - Restart your computer

2. **Install the Tool**
   ```bash
   cd C:\Users\steph\ad-review-tool
   npm install
   ```

3. **Get Claude API Key**
   - Go to https://console.anthropic.com/
   - Sign up and create an API key
   - Copy the key (starts with `sk-ant-`)

### Running the Tool

1. **Start the Server**
   ```bash
   cd C:\Users\steph\ad-review-tool
   npm start
   ```

2. **Open in Browser**
   - Go to http://localhost:3000
   - Paste your API key and click "Save API Key"

3. **Analyze Your Ads**
   - **Select your brand**: Marine Concepts or Coastline
   - **Select ad format**: Paid social, organic social, PPC, email, website, video, or offline
   - Upload one or multiple ad images
   - (Optional) Paste ad copy in the text box
   - Click "Get Elite Creative Director Analysis"
   - Wait 10-30 seconds for comprehensive feedback with exact rewrites

4. **Export Results**
   - Click "Export for Agency" to get a shareable HTML report
   - Click "Save to CSV" for data tracking

### When Done
- Press `Ctrl+C` in the command prompt to stop the server

## üîß Troubleshooting

### Server won't start
- Make sure Node.js is installed: `node --version`
- Run `npm install` again
- Check if port 3000 is already in use

### "Failed to fetch" error
- Make sure the server is running (`npm start`)
- Check that you're using http://localhost:3000

### API errors
- Verify your API key is correct
- Check your Anthropic account has credits
- Try with a smaller image file

### Slow analysis
- Multiple ads take longer (10-20 seconds each)
- Larger image files take more time
- This is normal‚Äîquality analysis takes time!
