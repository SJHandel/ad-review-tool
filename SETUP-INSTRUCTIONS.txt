ðŸ“‹ STEP 1: INSTALL NODE.JS (One-time setup)
--------------------------------------------------------------------------------
1. Go to: https://nodejs.org/
2. Download the "LTS" (Long Term Support) version
3. Run the installer and follow the prompts
4. Accept all defaults (just keep clicking "Next")
5. Restart your computer after installation

To verify Node.js is installed:
- Open Command Prompt (search for "cmd" in Windows)
- Type: node --version
- You should see something like "v20.x.x"


ðŸ“‹ STEP 2: INSTALL THE TOOL
--------------------------------------------------------------------------------
1. Open Command Prompt (search for "cmd" in Windows)

2. Navigate to the tool folder:
   cd C:\Users\steph\ad-review-tool

3. Install required packages:
   npm install

   (This will take 1-2 minutes - it's downloading required files)


ðŸ“‹ STEP 3: GET YOUR CLAUDE API KEY (One-time setup)
--------------------------------------------------------------------------------
1. Go to: https://console.anthropic.com/
2. Sign up or log in to your account
3. Click "Get API Keys" or "API Keys" in the menu
4. Click "Create Key"
5. Give it a name like "Ad Review Tool"
6. Copy the key (starts with "sk-ant-...")
7. IMPORTANT: Save this key somewhere safe - you won't be able to see it again!


ðŸ“‹ STEP 4: START THE TOOL
--------------------------------------------------------------------------------
1. Open Command Prompt

2. Navigate to the tool folder (if not already there):
   cd C:\Users\steph\ad-review-tool

3. Start the server:
   npm start

4. You should see a message saying "Server Running!"

5. Open your web browser (Chrome, Firefox, Edge, etc.)

6. Go to: http://localhost:3000

7. The tool should now be loaded!


ðŸ“‹ STEP 5: USE THE TOOL
--------------------------------------------------------------------------------

2. Click "Save API Key"
3. Upload an ad image (JPG or PNG)
4. Paste your ad copy (optional)
5. Click "Review Ad"
6. Wait 5-10 seconds for the analysis
7. Review the results
8. Click "Save Review to CSV" to save the results


ðŸ“‹ STOPPING THE TOOL

2. Press Ctrl+C
3. Type "Y" when asked to terminate
4. Close the Command Prompt window


ðŸ“‹ USING THE TOOL AGAIN LATER

ðŸ“‹ TROUBLESHOOTING
--------------------------------------------------------------------------------

Problem: "node is not recognized" error
Solution: Node.js is not installed. Go back to Step 1.

Problem: "Cannot find module" error
Solution: Run "npm install" again in the ad-review-tool folder.

Problem: "Port 3000 is already in use" error
Solution: Another program is using port 3000. Either:
  - Close other programs and try again, OR
  - Stop the previous instance by finding Command Prompt windows and closing them

Problem: API key error
Solution: Make sure your API key starts with "sk-ant-" and is copied correctly.

Problem: "Failed to fetch" error
Solution: Make sure the server is running (npm start) before opening the browser.
